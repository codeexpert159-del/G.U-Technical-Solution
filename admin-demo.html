ğŸ“š Integration Guide

To display content on your public pages, add these files to your HTML:

1. Add to <head>

<link href="assets/css/admin.css" rel="stylesheet">
<script src="assets/js/content-loader.js"></script>

2. Add Container

<div id="projectsContainer"></div>

3. Add JavaScript

<script>
window.addEventListener('contentLoaderReady', () => {
  const container = document.getElementById('projectsContainer');
  container.innerHTML = contentLoader.projects
    .map(p => contentLoader.renderProjectCard(p))
    .join('');
});
</script>
