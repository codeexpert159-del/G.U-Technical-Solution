================================================================================
                          KEY FILES EXPLAINED
================================================================================

admin/index.html
    - Main admin dashboard
    - Shows statistics (project & service count)
    - Quick links to manage content
    - Responsive navigation

admin/projects.html
    - Complete project management interface
    - Add new projects with form
    - Edit existing projects
    - Delete projects
    - Upload images and videos
    - Table view of all projects

admin/services.html
    - Complete service management interface
    - Add new services with form
    - Edit existing services
    - Delete services
    - Upload images and videos
    - Table view of all services

admin/api/get-projects.php
    - Returns all projects as JSON
    - Used by content loader to fetch data

admin/api/add-project.php
    - Creates new project
    - Saves to projects.json
    - Returns success/error response

admin/api/update-project.php
    - Updates existing project
    - Modifies projects.json
    - Preserves image/video if not changed

admin/api/delete-project.php
    - Removes project from database
    - Returns success/error response

admin/api/upload-project-media.php
    - Handles image and video uploads
    - Validates file types
    - Checks file size (max 100MB)
    - Stores in admin/uploads/projects/

admin/data/projects.json
    - JSON database for projects
    - Contains: id, title, category, description, 
      details, image, video, status, date

admin/data/services.json
    - JSON database for services
    - Contains: id, title, description, details,
      icon, image, video

assets/css/admin.css
    - Styling for admin interface
    - Styles for video players
    - Responsive layout styles
    - Media gallery styles

assets/js/content-loader.js
    - Fetches data from admin API
    - Renders project and service cards
    - Loads on page initialization
    - Dispatches 'contentLoaderReady' event

================================================================================
                          API ENDPOINTS REFERENCE
================================================================================

PROJECT APIs
GET  /admin/api/get-projects.php
     Returns: Array of all projects as JSON
     
POST /admin/api/add-project.php
     Requires: title, category, description, details, status
     Optional: image, video
     Returns: {success: true, id: number}
     
POST /admin/api/update-project.php
     Requires: id, and fields to update
     Returns: {success: true}
     
POST /admin/api/delete-project.php
     Requires: id
     Returns: {success: true}
     
POST /admin/api/upload-project-media.php
     Requires: file (multipart/form-data)
     Returns: {success: true, filename, path}

SERVICE APIs
GET  /admin/api/get-services.php
     Returns: Array of all services as JSON
     
POST /admin/api/add-service.php
     Requires: title, description, details
     Optional: icon, image, video
     Returns: {success: true, id: number}
     
POST /admin/api/update-service.php
     Requires: id, and fields to update
     Returns: {success: true}
     
POST /admin/api/delete-service.php
     Requires: id
     Returns: {success: true}
     
POST /admin/api/upload-service-media.php
     Requires: file (multipart/form-data)
     Returns: {success: true, filename, path}

STATISTICS
GET  /admin/api/get-stats.php
     Returns: {projectCount: number, serviceCount: number}
